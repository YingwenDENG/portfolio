3D Landscape Modeling


Introduction

Generate a consistent 2D model(ESRI datasets) of a urban section(Part A and B in the following image) in Dresden, Germany using ArcGIS, including entries to the feature attribute data based on the orthophoto. Feature attributes include type, land-use code, storey number and building height. Transform a copy of this GIS model into a different spatial reference system and 3D format. Refine the 3D model(Part B in the following image) with Blender, including 3D roof and facade construction, and associative texturing and scene creation.

2D Model

3D Model

Construct roofs, windows, doors and other attachments of the building. The mainly used tool in Blender is extrusion, translate, insert faces, scale, subdivide, knife and loop cut. Depending on the available resource, three methods are used to texture the model, which are "image texturing with other reference data", "image provided texturing", "mathematical method". And to spice up the model, external open 3D plants model are added. (Resource: http://free3d.com/) Snap tool used to guarantee the plant snap to the closest faces.
