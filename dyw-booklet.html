Booklet Design

A practice for realizing design concepts for print information products with DTP


Introduction

This is a desktop publishing (DTP) practice for the Geo-Media course. The booklet is specified to contain eight pages, size 6A, in portrait orientation. The aim of this project is to combine existing contents in a layout that looks professional.
This booklet about the EU history is created in May 2018 based on the contents on the website: europa.eu. (Some contents might be outdated now)
