An approach to detect semi-parallel line features

View on Github

Introduction

Design and implement a tool which detects semi parallel line structures using a pair wise comparison between features. The program shall break the original line features into parallel and non parallel sections. And during the comparison process, certain search distance in meters and tolerance of the parallel angle in degrees will be assigned.

Algorithm

The comparison works with breaking the polylines on the nodes in each polyline. And assign each segments with its vector direction in the XY coordinate system. Generating the near table of features within the search distance . Filter out the touching feature pairs. Compare the vect or directions of segments in both polyline by calculating the angle difference of the directions and select the ones within the angle tolerance and consider them as semi parallel segment pairs. And to complete the result table with the non parallel segments, compare with the original segment pairs and add missing IDs in each polylines with the responding “-1” value.

Output

The output attribute table shall fill the detect semi-parallel feature sections with the ID of segments pairs(as "OBJECTID"), the ID of segments in one polyline as "SegFC_ID" with the ID of its semi-parallel segments in the other polyline as "SegFC2_ID". For the segments with no semi-parallel segments, "-1" is given to the missing matching segment ID. Additionally, the semi-parallel segments are selected and highlighted in the data view.
